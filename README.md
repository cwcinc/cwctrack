# CWCTrack

CWCTrack is a modification of the popular game PolyTrack by Kodub. This mod is not officially endorsed by Kodub.

CWCTrack aims to provide many quality-of-life features that have been requested by the community in **#game-suggestions** and **#mod-suggestions** over the years. This is a compilation of my work bringing those requests to life.

## Releases

The latest version can always be found online at https://cwctrack.cwcinc.dev/.

For app downloads and installation instructions, read the [latest release](https://github.com/cwcinc/cwctrack/releases/latest).

## Features

### General

- Minimap
  - Visible while driving any track, displays car position
  - Displays all cars in a multiplayer room
  - Shown in editor to help builders create track thumbnail art

- Bookmarklet to export all tracks
  - Can be used to move all tracks from PolyTrack to CWCTrack or other versions
  - Found at bottom of settings
  - Drag to bookmarks bar and click while on https://app-polytrack.kodub.com/0.6.1/
  - After tracks are copied, paste them in the import box of any PolyTrack version

- Sort options for all tracks
  - Ascending / descending toggle
  - Sort by Save Time, Name, Author, Last Modified, Environment, ID
  - Click the list icon button to toggle track groups and folders

- Custom arrangements for custom tracks
  - Drag custom tracks to reorder them

- Displays time and date when each leaderboard record was set

- Realistic lighting mode
  - Sets the world to night time and gives PolyCar headlights

- Realistic crashing mode
  - Realistic screaming sounds when you crash hard

- Honk car horn (local)

- Disable all text italics option in settings

- Allows multiple game instances to be open simultaneously

- Devtools (F12) available in app for debugging

### Multiplayer

- Proximity chat
  - Mute/unmute microphone while in a room

- Custom hide and seek room type
  - Hides all car nametags and disables minimap for all players

### Editor

- Car trail shown with accurate hitbox representation after testing in editor
  - Decrease space between hitbox samples with the "Car Trail Spacing" setting

- CWCTrack exclusive track code format that reduces code size by ~50%

- Track setting to modify the creation date

### Racing

- Ranking symbol (grade) shown after a personal best
  - Assigns you a letter grade based on your proximity to the WR (S+ to F-)

- Offroad Particles / Skidmarks

- Live input visualizer
  - Shows input visualization while driving, not just in replays
  - Can be enabled in settings

- Backwards camera keybind
  - Option to enable it on hold or toggle

- Toggle ghost keybind

- Max ghost opacity setting
  - Decrease to make ghost less visible in runs
  - Applies to multiplayer cars as well

- Speedometer decimal places setting
  - Choose between 0-4 decimal places for your speedometer

- Fixed camera slider
  - Locks camera to the car when fully enabled
  - Default camera when disabled

- FOV multiplier setting
  - Change the rate of FOV changing as velocity increases
  - When at the lowest setting, camera FOV is completely static
